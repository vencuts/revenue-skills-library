# Daily Briefing Format Reference

## Design Principle

**30 seconds to read, actionable focus only.** No fluff, no status reports — just what you need to do and why.

---

## Priority Rules

| Signal | Impact on Briefing |
|---|---|
| Deal closing this week | 🔴 Top of Focus Today |
| External meeting today | Listed in Critical Meetings with deal context |
| Overdue task/follow-up | ⚠️ Needs Attention section |
| Unread Slack from key stakeholder | ⚠️ Needs Attention section |
| No meetings today | Frame as focus day with top priority task |
| Friday | Auto-switch to weekly summary |

## Morning Briefing — Full Example

```markdown
# ☀️ Morning Briefing — Monday, March 10, 2026

## Focus Today
1. Close out Acme Corp contract review — signing expected this week ($2.1M, Contracting)
2. Prep for TechCo discovery call at 2pm — first meeting, B2B Plus evaluation
3. Follow up on BetaShop launch blocker — Red health, API rate limit unresolved since Friday

## Critical Meetings
- 10:00am: Acme Corp — Contract Review ($2.1M, Contracting) 🔴
- 2:00pm: TechCo — Discovery Call ($800K, Qualify)
- 4:00pm: BetaShop — Launch Status (Case #4521, Red)

## ⚠️ Needs Attention
- GlobalWidgets: No activity in 12 days, close date is March 21
- Unread DM from Sarah (SE lead) re: Acme Corp pricing approval
- Overdue: Send TechCo pre-call questionnaire (due Friday)

## Action Items
- [ ] Review Acme Corp redline comments before 10am call
- [ ] Send TechCo pre-call questionnaire (overdue)
- [ ] Ping Product on BetaShop API rate limit escalation
```

## Standup — Full Example

```
**Yesterday:**
- Acme Corp: Submitted final contract redlines, awaiting legal review
- BetaShop: Escalated API rate limit to Product, opened support ticket
- Completed TechCo competitive research (vs BigCommerce)

**Today:**
- 10am: Acme Corp contract review call
- 2pm: TechCo discovery call (first meeting)
- Draft BetaShop launch risk assessment

**Blockers:**
- BetaShop API rate limit — waiting on Product team response (escalated Friday)
```

## Calendar Health Suggestions

These are **suggestions only** — never auto-book or modify calendar:

```
💡 Calendar Notes:
- No prep block before TechCo 2pm call — consider 30min at 1:30pm
- Back-to-back: Acme (10am) → Internal sync (11am) — no buffer
- BetaShop at 4pm is your last meeting — good time for follow-up emails after
```
