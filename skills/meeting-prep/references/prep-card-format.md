# Meeting Prep Card — Full Format Reference

## Card Template (External Meeting)

```markdown
## [🔴/🟡/🟢] [Meeting Title] - [Day]
**Time:** [Day, Date] at [Time] [Timezone]
**Attendees:**
- **[External Name]** ([email]) - [Role/Title] @ [Company]
- [Shopify Attendee] ([email]) - [Role]

### Deal Context
- **Opportunity:** [Opp Name]
- **Stage:** [Stage] | **Probability:** [X%]
- **Amount:** $[X] | **Close Date:** [Date]
- **Primary Product:** [Plus/POS/Markets/etc.]
- **AE:** [Name] | **SE:** [Name] | **CSM:** [Name]
- **Days in Stage:** [N]
- **Competitor:** [If known]

### Account Overview
- **Industry:** [Industry]
- **Current Platform:** [If known]
- **GMV/Revenue:** [If available]
- **Key Pain Points:** [From discovery or prior calls]

### Recent Activity (Last 30 Days)
**Email Threads:**
- [Date]: [Subject] — [Key points, action items, decisions]
- [Date]: [Subject] — [Key points]

**Slack Mentions:**
- [Date] in #[channel]: [Summary]

**Salesloft Calls:**
- [Date]: [Call title] ([Duration]) — [Summary highlights, objections, commitments]

**Fellow Notes:**
- [Date]: [Meeting title] — [Key decisions, action items with owners]

### Open Questions & Commitments
**You promised to follow up on:**
- [ ] [Item from prior call/email]
- [ ] [Item]

**They committed to:**
- [ ] [Item — e.g., "Send current tech stack diagram"]

### Suggested Prep
**Topics to Cover:**
- [Topic] — [Why it matters: context from deal/prior calls]
- [Topic] — [Context]

**Questions to Ask:**
- [Question grounded in deal context]
- [Question]

**Materials to Have Ready:**
- [Document/demo/link]

### Competitive Intel
[If competitor known — key positioning points, S.T.A.R. objection handling]

### Gaps & Risks
**Missing Information:**
- [What you don't know that you should]

**Risk Signals:**
- [Deal risk indicators: stalled stage, no recent activity, competitor mentioned]

**Pre-Call Action Items:**
- [ ] [Research X before the call]
- [ ] [Prepare demo of Y]
```

## Priority Rules

| Condition | Priority |
|---|---|
| Meeting is today | 🔴 |
| Meeting is tomorrow | 🟡 |
| Meeting is 2+ days out | 🟢 |
| High-value deal (>$500K) closing this quarter | Bump up one level |
| No recent activity (>30 days) on active opp | Flag ⚠️ |

---

## Slack Channels for Additional Context

Check these channels when relevant context might exist:
- `#help-[team]` — Team-specific support threads
- `#sales-ops` — Deal status, process questions
- `#competitive` — Competitor mentions, win/loss
- `#product-updates` — Recent platform changes relevant to merchant

## Tips for Efficient Prep

| Depth | Actions | Time |
|---|---|---|
| **Quick** (🟢) | SF opp context + last email thread | ~1 min |
| **Standard** (🟡) | + Slack + transcripts + Drive docs | ~3 min |
| **Deep** (🔴) | + Vault product updates + Fellow notes + competitive intel + prior TAs | ~5 min |
